## Установка

1. Установите библиотеку Pillow & random:
    ```
    pip install Pillow
    pip install random
    ```

2. Скопируйте этот репозиторий.
3. Запустите main.py:
    ```
    python main.py
    ```
4. Зайдите в папку "avatars"
5. Вы великолепны
